constructor

Create a new JSON database or initialize an exisiting database.

constructor(database: string): boolean
Parameters
database (string) The name of the JSON database to be created or used.
Returns
boolean: Whether an existing JSON file was used or created or the action failed.
Example
const jsoning = require('jsoning');
var database = new jsoning("database.json");

set

Adds an element to a database with the specified value. If element exists, element value is updated.

set(key: string, value: any): boolean
Parameters
key (string) Key of the element to be set.
value (any) Value of the element to be set.
Returns
boolean: If element is set/updated successfully, returns true, else false.
Example
database.set("foo", "bar");
database.set("hi", 3);

database.set("en", "db"); // { "en": "db" }
database.set("en", "en"); // { "en": "en" }

let set = database.set("khaleel", "gibran");
console.log(set); // returns true

all

Returns all the elements and their values of the JSON database.

all(): Object
Returns
Object: The object of all the key-value pairs of the database.
Example
database.set("foo", "bar");
database.set("hi", "hello");

let all = database.all();
console.log(all); // { "foo": "bar", "hi": "hello" }

delete

Delete an element from the database based on its key.

delete(key: string): Boolean
Parameters
key (string) The key of the element to be deleted.
Returns
Boolean: Returns true if the value exists, else returns false.
Example
database.set("ping", "pong");
database.set("foo", "bar");

database.delete("foo"); // returns true

get

Gets the value of an element based on it's key.

get(key: string): any
Parameters
key (string) The key of the element to be fetched.
Returns
any: Returns value, if element exists, else returns false.
Example
database.set("food", "pizza");

let food = database.get("food");
console.log(food) // returns pizza

math

Performs mathematical operations on values of elements.

math(key: string, operation: string, operand: number): Boolean
Parameters
key (string) The key of the element on which the mathematical operation is to be performed.
operation (string) The operation to perform, one of add, subtract, multiply and divide.
operand (number) The number for performing the mathematical operation (the operand).
Returns
Boolean: True if the operation succeeded, else false.
Example
database.set("value1", 1);
database.set("value2", 10);

database.math("value1", "add", 1);
database.math("value2", "multiply", 5);

console.log(database.get("value1")); // returns 1+1 = 2
console.log(database.get("value2")); // returns 10*5 = 50

has

See if a particular element exists by using it's key.

has(key: string): Boolean
Parameters
key (string) The key of the element to see if the element exists.
Returns
Boolean: True if the element exists or false if the element doesn't exist.
Example
database.set("some value", "hi");

let has = database.has("some value");
console.log(has); // returns true 

let has2 = database.has("value");
console.log(has2); // returns false
